Privacy Policy

This Privacy Policy outlines the data practices of the Salesforce OmniStudio Network Logger Chrome extension.

Information Collection

The extension does not collect, store, or share any personal data, browsing history, or network request details outside your local environment.

How the Extension Works

The extension logs and displays network requests initiated by OmniStudio components (such as OmniScripts, FlexCards) to aid in debugging. This includes:

  • Integration Procedure calls
  • Data Raptor requests
  • Remote Actions

All logged data remains within your browser and is never sent to any external servers.

Copy to Clipboard Feature

The extension provides a "Copy to Clipboard" feature for convenience. Data copied using this feature is only stored in your system's clipboard and is not accessed or sent by the extension.

Permissions

The extension requires access to network requests to log and display relevant information for debugging purposes. This access is limited to the scope of the OmniStudio components in Salesforce.

No Data Sharing

The extension does not transmit or share any data with third parties. All functionality is executed locally within your browser.

Changes to This Policy

This privacy policy may be updated occasionally. You are encouraged to review this page periodically for any changes.
